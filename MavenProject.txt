Maven Project:-
Maven is a powerful project management tool that is based on POM (project object model). 
It is used for projects build, dependency and documentation.

Project Object Model (POM) file

There are two way to generate Maven Project:

1. By using command prompt
	a. mvn archetype:generate
    b.mvn archetype:generate -DgroupId=groupid -DartifactId=artifactid -DarchetypeArtifactId=maven-archetype-quickstart -DinteractiveMode=booleanValue

2. By using IDEs
